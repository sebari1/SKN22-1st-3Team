# Docs Overview

이 폴더에는 대시보드/ETL 프로젝트 관련 협업 문서를 모아 두었다. 용도별 요약은 다음과 같다.

- `project_summary.md` – 프로젝트 목표, 주요 기능, 데이터 흐름 요약.
- `project_roadmap.md` – 릴리즈/스프린트 계획과 Todo.
- `project_issues.md` – 운영 중 발견된 이슈 및 추적 기록.
- `etl_planning.md` – ETL 전체 아키텍처, 단계별 설명, admin 페이지 버튼 구성의 근간.
- `page_specification.md` – Streamlit 페이지별 목적/구성/데이터 출처 명세.
- `db_schema_and_erd.md`, `erd.md` – DB 스키마 설명과 ER 다이어그램.
- `word_definition.md` – 용어 정의/약어 표.
- `naver_blog_wordcloud_feature.md` – 블로그/워드클라우드 기능 요구사항.
- `project_structrue.md` – (오타 유지) 저장소 디렉터리 구조 문서.
- `wireframe_main.png`, `wireframe_pages.png` – 메인/서브 페이지 와이어프레임.

필요한 문서를 추가하거나 수정할 경우 이 README도 함께 갱신해 주면 빠른 onboard에 도움이 된다.
